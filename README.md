## Layers

- name

bicycle:
- highway (type)
- name (of the highway)
- cycleroute (info about highest ranking `roude=bicycle` on this highway; values: icn, rcn, lcn)
- ref (refs of all bicycle routes)
- surface

contours
- height (of the contour in meters)
